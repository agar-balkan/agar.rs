    • Agar Balkan Open Source

    Game description

    The Balkan version of the online fun game agar.io. The goal in this game is to eat food to increase your mass, but be careful not to be eaten by larger players.

    Servers

    FFA - A classic game with an old fashioned ogre mode. The size of the map is 20,000x20,000.

    MINIONS - A game in which each player gets 5 minions that follow him. The player can switch control between himself and the minion by pressing the Q button.

    INSTANT MERGE - A server where you merge as quickly as possible.

    MEGA SPLIT - A server on which it is possible to split into 64 parts.

    EXPERIMENTAL - Server with motherscell and smooth splits.

    RAINBOW - A server that changes the color of players.

    VIRUS MINES - A server where you bombard your opponents with viruses.

    OLD EXP - A classic, old-fashioned server with experimental mode.

    TEAMS - Division into 3 teams - red, blue, green.

    TOURNAMENT - The server where the tournaments are played. 6 players are required to start the game.
